# Metal Surface Defect Detection

## Overview
This project implements a Convolutional Neural Network (CNN) to automatically detect and classify metal surface defects. The workflow includes image preprocessing, model training, and evaluation to improve quality inspection in manufacturing processes.

## Technologies Used
- Python
- TensorFlow 
- Keras
- NumPy
- Matplotlib
- Pandas

## Dataset
NEU Surface Defect Dataset (publicly available), containing images of metal surfaces with multiple defect types.

## ML Approach
- Deep Learning
- Supervised Learning
- Image Classification using CNN

## Results
The CNN model accurately detects and classifies defects, improving automation in quality control.

## Future Improvements
- Experiment with deeper architectures (ResNet, EfficientNet)
- Deploy as a web app using Flask or Streamlit
- Implement real-time defect detection on production lines
